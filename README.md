# Language model training

## Set up

```bash
$ python -m venv env
$ source env/bin/activate
$ pip install --upgrade pip -i https://pypi.tuna.tsinghua.edu.cn/simple/
$ pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/
```
